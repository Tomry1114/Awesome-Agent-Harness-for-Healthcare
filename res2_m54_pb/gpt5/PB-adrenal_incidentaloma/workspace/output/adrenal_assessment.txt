B. Screen for pheochromocytoma
- Order: plasma free metanephrines.
- Rationale: adrenal mass plus palpitations/SVT and hypertension; lesion is not yet fully characterized with adrenal protocol imaging.
- Preparation/instructions:
  - Prefer fasting morning draw if feasible.
  - Rest supine quietly for 20–30 minutes before blood draw.
  - Avoid caffeine, nicotine, vigorous exercise, and acute stress/illness before testing if possible.
  - If clinically feasible, review medications that can interfere with interpretation; if not possible, interpret results with caution.

C. Screen for primary aldosteronism
- Order: plasma aldosterone concentration and plasma renin activity or direct renin with aldosterone/renin ratio.
- Rationale: hypertension requiring 2 agents, with intermittent elevated readings. Normal potassium does not exclude primary aldosteronism.
- Preparation/instructions:
  - Draw in the morning after the patient has been ambulatory for a period and seated per local lab protocol.
  - Ensure adequate/normal sodium intake before testing.
  - Correct hypokalemia before testing if present (not currently present).
  - Interfering antihypertensives can alter results; the patient is currently on irbesartan and amlodipine. If BP allows, medication adjustment may improve test accuracy, but if washout is unsafe, testing can still be obtained and interpreted cautiously.
  - If ARR is positive, proceed with confirmatory testing per local protocol unless clinical phenotype is clearly diagnostic.

Contingency plan / interpretation guidance
1) If plasma free metanephrines are elevated
- Repeat/confirm as needed depending on degree of elevation and pretest conditions.
- Urgent endocrinology follow-up and adrenal imaging review.
- If markedly elevated, treat as possible pheochromocytoma; initiate appropriate alpha blockade and coordinate specialty referral before any invasive procedure.

2) If aldosterone/renin ratio is elevated
- Review potassium, sodium intake, and medication effects.
- Proceed with confirmatory testing if indicated.
- If primary aldosteronism is confirmed, manage with mineralocorticoid receptor antagonist or subtype evaluation (adrenal venous sampling if surgical candidacy and lateralization are being considered).

3) If 1 mg DST cortisol is not suppressed
- Evaluate for mild autonomous cortisol secretion (MACS): check ACTH, consider repeat DST or additional cortisol assessment per local protocol.
- Assess and optimize cortisol-related comorbidities (hypertension, glucose intolerance, bone health).
- Consider adrenalectomy only if there is convincing biochemical autonomy and clinically meaningful comorbidity burden, especially if imaging remains unilateral and surgical risk is acceptable.

4) If MRI confirms benign adenoma and hormonal testing is negative
- Conservative management with interval clinical follow-up.
- Repeat imaging only if recommended by the imaging phenotype or if there is interval hormonal change/symptom evolution.
